## vtk2DPolyGenerator.cpp
Once you have the executable run it with these arguments: <br>
```
./exeutableName a b c d h
```
<br>
where, <br>
a = Origin of X <br>
b = Origin of Y <br>
c = Max X <br>
d = Max Y <br>
h = Number of divisions (Refining) <br>
*ONLY WORKS FOR SQUARE DOMAIN IN FIRST QUADRANT* <br>
